## Sample Flashcards

**Card 1:** What is the difference between list and tuple?
**Answer:** List is mutable, tuple is immutable. Lists use [], tuples use (). Lists slower, use more memory. Tuples can be dict keys.

**Card 2:** Explain GIL
**Answer:** Global Interpreter Lock. Mutex protecting Python objects. Only one thread executes bytecode at time. Affects CPU-bound tasks, not I/O-bound.
