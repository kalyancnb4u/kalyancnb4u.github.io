## Books

### Beginner
1. **The Go Programming Language** - Donovan & Kernighan
   - The definitive Go book
   - Comprehensive coverage
   - Clear examples

2. **Head First Go** - Jay McGavren
   - Beginner-friendly
   - Visual learning
   - Hands-on approach

3. **Get Programming with Go** - Nathan Youngman
   - Step-by-step lessons
   - Project-based
   - Good for beginners

### Intermediate
4. **Concurrency in Go** - Katherine Cox-Buday
   - Deep dive into concurrency
   - Patterns and best practices
   - Production examples

5. **Go in Action** - Kennedy, Ketelsen, St. Martin
   - Real-world applications
   - Best practices
   - Production techniques

6. **Learn Go with Pocket-Sized Projects** - Pac ktPub
   - Practical projects
   - Hands-on learning
   - Modern Go features

### Advanced
7. **Distributed Services with Go** - Travis Jeffery
   - Build distributed systems
   - Production patterns
   - Real-world architecture

8. **Cloud Native Go** - Hoffman & Choi
   - Microservices
   - Kubernetes
   - Cloud deployment

9. **Black Hat Go** - Steele & Kottmann
   - Security applications
   - Network programming
   - Offensive security
