### **Week 1: Fundamentals**

**Day 1: FizzBuzz**
Write a function that prints numbers 1-100. For multiples of 3 print "Fizz", multiples of 5 print "Buzz", multiples of both print "FizzBuzz".

**Day 2: Palindrome Checker**
Write a function to check if a string is a palindrome (reads same forwards/backwards).

**Day 3: Reverse Array**
Reverse an array without using `reverse()` method.

**Day 4: Find Duplicates**
Find duplicate elements in an array.

**Day 5: Anagram Checker**
Check if two strings are anagrams.

**Day 6: Sum of Array**
Calculate sum of all numbers in nested arrays.

**Day 7: Object Deep Clone**
Implement deep cloning of objects.
