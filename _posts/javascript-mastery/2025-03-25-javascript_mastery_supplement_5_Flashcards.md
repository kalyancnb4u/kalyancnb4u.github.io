## ðŸŸ¢ Fundamentals (Cards 1-50)

### **Card 1:**
**Q:** What are the 7 primitive data types?
**A:** string, number, boolean, null, undefined, symbol, bigint

### **Card 2:**
**Q:** Difference between `let`, `const`, and `var`?
**A:** `let` is block-scoped, can reassign. `const` is block-scoped, cannot reassign. `var` is function-scoped, can redeclare (avoid).

### **Card 3:**
**Q:** What are falsy values in JavaScript?
**A:** false, 0, "", null, undefined, NaN

### **Card 4:**
**Q:** What is hoisting?
**A:** JavaScript moves variable and function declarations to top of scope before execution.

### **Card 5:**
**Q:** Difference between `==` and `===`?
**A:** `==` does type coercion, `===` checks type AND value (strict equality).

## ðŸŸ¡ Advanced (Cards 51-100)

### **Card 51:**
**Q:** Explain the event loop
**A:** Event loop handles async operations by checking call stack, then microtask queue (promises), then macrotask queue (setTimeout).

### **Card 52:**
**Q:** What is a closure?
**A:** A function that remembers variables from its outer scope even after outer function has returned.

### **Card 53:**
**Q:** Promise states?
**A:** Pending, Fulfilled, Rejected

### **Card 54:**
**Q:** Difference between `map()` and `forEach()`?
**A:** `map()` returns new array, `forEach()` returns undefined, used for side effects.

### **Card 55:**
**Q:** What is `this` in arrow functions?
**A:** Lexically bound (inherited from parent scope), not its own `this`.

## ðŸ”´ Expert (Cards 101-150)

### **Card 101:**
**Q:** Explain prototypal inheritance
**A:** Objects inherit properties/methods from prototype chain via `[[Prototype]]`

### **Card 102:**
**Q:** What is event delegation?
**A:** Attaching single event listener to parent element instead of multiple children using event bubbling.

### **Card 103:**
**Q:** Explain debounce vs throttle
**A:** Debounce waits for pause (last call wins). Throttle executes at intervals (first in window wins).

### **Card 104:**
**Q:** What is memoization?
**A:** Caching function results based on inputs to avoid recomputation.

### **Card 105:**
**Q:** Shallow vs deep copy?
**A:** Shallow copies references, deep copies nested objects recursively.

## ðŸŽ¯ Interview-Specific (Cards 151-200)

### **Card 151:**
**Q:** Time complexity of array methods?
**A:** `push/pop`: O(1), `shift/unshift`: O(n), `indexOf/includes`: O(n), `sort`: O(n log n)

### **Card 152:**
**Q:** How to avoid callback hell?
**A:** Use Promises, async/await, or modularize functions.

### **Card 153:**
**Q:** What is CORS?
**A:** Cross-Origin Resource Sharing - browser security restricting cross-origin HTTP requests.

### **Card 154:**
**Q:** Explain XSS
**A:** Cross-Site Scripting - injecting malicious scripts. Prevent: sanitize input, escape output, CSP.

### **Card 155:**
**Q:** What is tree shaking?
**A:** Removing unused code from bundles during build process.
